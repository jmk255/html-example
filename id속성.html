id 속성

요소의 식별자로 사람에게 아이디를 부여하듯 요소에 아이디를 부여하면, 브라우저는 해당 요소를 정확하고 신속하게 식별할 수 있다.

id 속성값은 문서 내에서 유일해야 하며, 문자, 숫자, 하이픈(-), 언더스코어(_) 등을 사용하여 값을 작성할 수 있다.

id 작성시 주의 사항

  1. 영문자 소문자로 시작해야 합니다
  2. 숫자로 시작할 수 없습니다
  3. 대부분 특수 문자는 사용할 수 없지만 $,_는 사용할 수 있고 첫 문자로도 사용이 가능합니다
  4. 공백 문자는 사용할 수 없습니다
